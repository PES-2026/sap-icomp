# 🐞 Template de Relatório de Bugs (Bug Report)

Use este modelo sempre que um teste falhar ou um erro for encontrado em qualquer ambiente (**Dev**, **Test** ou **Main**).

### 🏷️ Título do Bug
`[MODULO] Descrição curta e clara do erro`  
*Ex: [Login] Erro 500 ao tentar recuperar senha com e-mail inexistente.*

---

### 📝 Descrição
Uma explicação rápida do que está acontecendo e qual o impacto (ex: o usuário não consegue finalizar a compra).

### 🔄 Passos para Reproduzir
1. Acesse a URL `https://...`
2. Clique no botão "X".
3. Preencha o campo "Y" com o valor "123".
4. Observe o erro que aparece na tela.

### ❌ Comportamento Atual
*O que o sistema está fazendo agora?* Ex: A tela fica branca e o console do navegador mostra um erro de "undefined".

### ✅ Comportamento Esperado
*O que o sistema deveria fazer?* Ex: O sistema deveria exibir uma mensagem amigável dizendo "E-mail não encontrado".
